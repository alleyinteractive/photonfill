#PhotonFill

PhotonFill is a WordPress plugin that allows for easy implementation of responsive images using [picturefill](https://github.com/scottjehl/picturefill) while integrating the powerful free CDN of [Jetpack Photon](https://jetpack.me/support/photon/). This plugin optionally allows the implementation of lazy loaded responsive images using [lazysizes](https://github.com/aFarkas/lazysizes).
